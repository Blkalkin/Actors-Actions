# Frontend

React + TypeScript + Vite frontend for the world simulation.

## Setup

```bash
npm install
npm run dev
```

Runs at `http://localhost:3000`

## Environment

Create `frontend/.env` if needed:

```bash
VITE_API_URL=http://localhost:8000
```

Default proxies `/api` requests to `localhost:8000` (see `vite.config.ts`)
